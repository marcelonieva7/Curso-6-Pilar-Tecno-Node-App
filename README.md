
# Covid Info Center

Proyecto backend para el curso final de PilarTecno 2021.

Api que brinda informacion sobre los centros de vacunacion en La Rioja.


## Tech Stack

- Node.js
- Express.js
- Mongo BD
- Mongoose
- Joi.js
- Passport.js
